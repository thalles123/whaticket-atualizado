# sistema
 
FAZENDO DOWNLOAD DO INSTALADOR & INICIANDO A PRIMEIRA INSTALAÇÃO (USAR SOMENTE PARA PRIMEIRA INSTALAÇÃO):

```bash
sudo apt install -y git && git clone https://github.com/thalles123/whaticket-atualizado && sudo chmod -R 777 sistema && cd sistema && sudo ./install_primaria
```

```bash
cd ./sistema && sudo ./sistema
```
